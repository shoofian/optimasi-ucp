Ini adalah instruksi bagaimana menjalankan program "Optimasi Use Case Point dengan Particle Swarm Optimization"

1. Siapkan Python 3

2. Install Pandas
	- Buka CMD pada jendela baru
	- ketik "pip install pandas"
	- Tunggu hingga selesai

3. Install Openpyxl
	- Pada jendela CMD yang sama saat menginstall Pandas
	- ketik "pip install openpyxl"
	- Tunggu hingga selesai

4. Install Pyswarm
	- Pada jendela CMD yang sama saat menginstall Pandas dan Openpyxl
	- ketik "pip install pyswarms"
	- Tunggu hingga selesai
